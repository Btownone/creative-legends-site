# creative-legends-site

Bridget Farhat — **Creative Legends Co.** — photo + video studio. Single-page static site.

**Live:** https://creative-legends-site.onrender.com (custom domain: creativelegends.co — DNS cutover pending; existing site currently serves from creativelegends.co)

**Stack:** One file. `index.html`. Inline CSS + JS. No build step. No framework.

## Editing

Use Claude Code:

```bash
git clone https://github.com/Btownone/creative-legends-site
cd creative-legends-site
claude
```

Then tell it what to change — "swap the hero image", "add a new testimonial", "change the about copy". Claude edits `index.html`, commits, pushes. Render auto-deploys in ~30s.

See `CLAUDE.md` for the AI editing guide.
